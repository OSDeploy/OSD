# Get-OSDCloudOSNames

## SYNOPSIS
Returns the Operating Systems names used by OSDCloud

## SYNTAX

```
Get-OSDCloudOSNames [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Returns the Operating Systems names used by OSDCloud

## EXAMPLES
