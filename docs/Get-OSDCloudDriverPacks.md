# Get-OSDCloudDriverPacks

## SYNOPSIS
Returns the DriverPacks used by OSDCloud

## SYNTAX

```
Get-OSDCloudDriverPacks [<CommonParameters>]
```

## DESCRIPTION
Returns the DriverPacks used by OSDCloud

## EXAMPLES

### EXAMPLE 1
```
Get-OSDCloudDriverPacks
```

Returns all OSDCloud driver packs from the module catalog.
