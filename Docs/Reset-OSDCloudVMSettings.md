# Reset-OSDCloudVMSettings

## SYNOPSIS
Resets OSDCloudVM to its default settings stored in the OSDCloud Template and current Workspace

## SYNTAX

```
Reset-OSDCloudVMSettings [<CommonParameters>]
```

## DESCRIPTION
Resets OSDCloudVM to its default settings stored in the OSDCloud Template and current Workspace

## EXAMPLES

### EXAMPLE 1
```
Reset-OSDCloudVMSettings
```
