# Update-OSDCloudUSB

## SYNOPSIS
Updates an OSDCloud USB by downloading OS and Driver Packs from the internet

## SYNTAX

```
Update-OSDCloudUSB [[-DriverPack] <String[]>] [-PSUpdate] [-OS] [[-OSLanguage] <String>]
 [[-OSActivation] <String>] [[-OSName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Updates an OSDCloud USB by downloading OS and Driver Packs from the internet
